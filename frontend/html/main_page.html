              Добавить прописанного
              1. Нажмите на иконку "add" в панели "Прописанные";
              2. Заполните все необходимые поля в форме;
              3. Нажмите на кнопку "Добавить".
              Редактировать прописанного
              1. Нажмите на иконку "edit" в таблице "Прописанные";
              2. Отредактируйте все необходимые поля в форме;
              3. Нажмите на кнопку "Сохранить".
              Выполнить справку
              1. Нажмите на кнопку "event_note" в таблице "Прописанные";
              2. Выберите из списка нужную справку;
              3. Укажите период для выполнения справки (если требуется);
              4. Нажмите на кнопку "Выполнить".
              Добавить контакт
              1. Нажмите на иконку "call" в панели "Дополнительная информация";
              2. Нажмите на вкладку "Добавить";
              3. Заполните все необходимые поля в форме;
              4. Нажмите на кнопку "Добавить".
              Редактировать контакт
              1. Нажмите на иконку "call" в панели "Дополнительная информация";
              2. Нажмите на вкладку "Редактировать";
              3. Выберите контакт из списка контактов;
              4. Заполните все необходимые поля в форме;
              5. Нажмите на кнопку "Сохранить".
              1. Нажмите на кнопку "Изменить";
              2. Заполните поле "Примечание для печати";
              3. Нажмите на кнопку "Сохранить".
